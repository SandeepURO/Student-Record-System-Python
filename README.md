# Student Record System 🎓
Built with pure Python — no libraries.

## Features
- ➕ Add students
- 📋 Show all students
- 🔍 Search by name
- ❌ Delete students
- 📊 Class statistics
- 💾 Data saved to REGISTER.txt

## Tech
- Python 3
- Jupyter Notebook

## Concepts Used
- Functions, Lists, Dictionaries
- Loops, Conditionals
- File Handling

## How to Run
1. Open Jupyter Notebook
2. Run Stage1_Day9_MiniProject.ipynb
